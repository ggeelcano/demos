# Foto: sala-grande

Sala grande con vigas vista. Va en formato ancho (2 columnas).

## Cómo añadir tu foto

1. Renombra tu foto a **`foto.jpg`** (o `.png` / `.jpeg`).
2. Arrástrala a esta carpeta.
3. Recarga la web — aparece sola.

Si está vacía, la web muestra un placeholder con la "M" del Motabarri.
