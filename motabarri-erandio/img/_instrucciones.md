# Cómo poner las fotos del Motabarri

Cada subcarpeta corresponde a un hueco de la web. Mete tu foto con el nombre **`foto.jpg`** (o `.png` / `.jpeg`) dentro de la carpeta correspondiente. La web la coge sola.

| Carpeta | Hueco en la web | Tipo |
|---|---|---|
| `portada/` | Hero de inicio + foto grande de la galería | Exterior, panorámica |
| `exterior/` | Vista lateral del caserío | Exterior |
| `comedor/` | Comedor principal | Interior |
| `sala-grande/` | Sala con vigas y piedra | Interior, ancho |
| `sala-intima/` | Sala íntima planta baja | Interior |
| `mesas/` | Mesas y candelabros | Detalle |
| `carpa/` | Carpa con chimenea | Interior carpa |
| `terraza/` | Terraza al aire libre | Exterior |
| `txuleton/` | Txuletón a la brasa | Plato |
| `bacalao/` | Bacalao al pil-pil | Plato |
| `alubias/` | Alubias con sacramentos | Plato |
| `brasa/` | Brasa encendida (ancho) | Detalle ancho |

## Reglas
- **Nombre del archivo:** siempre `foto.jpg` (o `foto.png` o `foto.jpeg`).
- **Tamaño mínimo recomendado:** 1200 px de ancho.
- **Formato:** JPG o PNG, color (no blanco y negro).
- **Si la carpeta está vacía** → la web muestra un placeholder con la "M" del Motabarri.
- **Cuando metes una foto** → aparece automáticamente al recargar la página.
