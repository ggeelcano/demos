# Foto: portada

Foto principal del caserío. Va de fondo en el hero (lo primero que se ve) y en la galería como pieza grande 2x2. Mejor en horizontal y panorámica.

## Cómo añadir tu foto

1. Renombra tu foto a **`foto.jpg`** (o `.png` / `.jpeg`).
2. Arrástrala a esta carpeta.
3. Recarga la web — aparece sola.

Si está vacía, la web muestra un placeholder con la "M" del Motabarri.
